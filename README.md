# Netflix-Data-Analysis-Python
Netflix Data Analysis using Python, Pandas and Matplotlib with EDA dashboards and business insights.
 
# 📌 Project Overview

This project analyzes the Netflix Titles dataset using Python.

The project demonstrates the complete Data Analysis workflow:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Dashboard Creation
- Business Insights

# 📈 Key Findings

✅ Movies account for **69.7%** of Netflix content.

✅ TV Shows account for **30.3%**.

✅ United States contributes the highest number of titles.

✅ Top 10 countries contribute **78.01%** of all Netflix content.

✅ TV-MA is the most common rating.

✅ International Movies is the most popular genre.
